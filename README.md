# **ATS Website Redesign**

This repository contains the **front-end redesign** of the [Altius Tech Solutions Inc.](https://altiustechsolutions.com/) website.  
The goal is to improve the **overall visual design, layout, and user experience** while keeping the existing backend functionality intact.

> **Note:**  
> This repository contains *only front-end files* (HTML, CSS, and client-side JavaScript).  
> No backend logic or sensitive data (such as PHP, Python, or database files) are included.

---

## **Clone Purpose**

This redesign focuses purely on:
- Enhancing the layout and structure using modern standard CSS practices.
- Refining animations, icons, and user interface components.
- Introducing new visual sections and user options.
- Implementing a **Light/Dark mode toggle**.
- Applying design improvements for a cleaner, more intuitive look.
